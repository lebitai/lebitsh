快速开始

只需一行命令即可安装

curl --proto '=https' --tlsv1.2 -sSf i.lebit.sh | sh
已复制!

上面的命令将下载并运行交互式安装程序，您可以选择需要的模块进行安装。

也可以直接安装特定模块:

curl --proto '=https' --tlsv1.2 -sSf https://docker.lebit.sh | sh
已复制!

或者使用替代方式:

curl --proto '=https' --tlsv1.2 -sSf i.lebit.sh | sh -s -- docker
已复制!
