Installation

Get started with Lebit.sh in seconds.

Install with one command

Run this in your terminal to install Lebit.sh with all modules:

Terminal
curl -sSf https://lebit.sh/install | sh

This command will download and run the installation script, which will:

  • Check your system requirements
  • Download the latest version of Lebit.sh
  • Install it to /usr/local/bin/lebit
  • Set up configuration files

Install a specific module

If you only need specific functionality, you can install individual modules:

Terminal
curl -sSf https://lebit.sh/install | sh -s -- system

Available modules:

  • System Management: curl -sSf https://lebit.sh/install | sh -s -- system
  • Docker Management: curl -sSf https://lebit.sh/install | sh -s -- docker
  • Development Environment: curl -sSf https://lebit.sh/install | sh -s -- dev
  • System Tools: curl -sSf https://lebit.sh/install | sh -s -- tools
  • Mining Tools: curl -sSf https://lebit.sh/install | sh -s -- mining

Quick Alias Setup

Enhance your terminal productivity with our collection of useful aliases:

Terminal
curl -s https://lebit.sh/alias | bash

This command will set up useful aliases in your shell configuration. Learn more.

Manual installation

If you prefer to install manually:

  1. Download the latest release from GitHub
  2. Extract the archive
  3. Run the install script: sudo ./install.sh
