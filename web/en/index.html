Installation

Start using Lebit.sh in just one step

curl --proto '=https' --tlsv1.2 -sSf https://lebit.sh | sh
Copied!

This command will download and execute the installation script, which will guide you through the setup process.

Compatibility

Ubuntu
CentOS
Fedora
Debian
Arch Linux

You can also install specific modules directly:

curl --proto '=https' --tlsv1.2 -sSf https://docker.lebit.sh | sh
Copied!

Or use the alternative approach:

curl --proto '=https' --tlsv1.2 -sSf https://lebit.sh | sh -s -- docker
Copied!
