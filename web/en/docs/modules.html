Modules

Lebit.sh is designed with a modular architecture, allowing you to install and use only the components you need. Each module serves a specific purpose and can be installed independently. This page provides detailed information about each module, including their functionalities and available commands.

System Management

System Management Module

The system management module provides tools for monitoring, optimizing, and maintaining your Linux system. It includes commands for gathering system information, cleaning unnecessary files, and optimizing system performance.

Key Commands:
lebit system info
Displays comprehensive information about your system, including CPU, memory, disk, and operating system details.
lebit system clean
Cleans up system logs, temporary files, and package caches to free up disk space.
lebit system optimize
Optimizes system settings for better performance based on your hardware configuration.
lebit system monitor
Real-time monitoring of system resources, including CPU, memory, and disk usage.

Docker Management

Docker Management Module

The Docker management module simplifies the installation, configuration, and management of Docker and Docker Compose. It provides commands for managing containers, images, volumes, and networks.

Key Commands:
lebit docker install
Installs Docker and Docker Compose on your system with optimal settings.
lebit docker upgrade
Upgrades Docker and Docker Compose to the latest versions while preserving your configurations.
lebit docker prune
Cleans up unused Docker resources, including containers, images, volumes, and networks.
lebit docker stats
Displays resource usage statistics for running Docker containers.

Development Environment

Development Environment Module

The development environment module helps you set up a complete development environment for various programming languages and frameworks. It includes tools for installing programming languages, development tools, and code editors.

Key Commands:
lebit dev install [language]
Installs a specific programming language and its development tools. Supported languages include Python, Node.js, Go, Ruby, and more.
lebit dev editor [editor]
Installs and configures a code editor. Supported editors include VSCode, Sublime Text, Vim, and Emacs.
lebit dev setup [framework]
Sets up a development environment for a specific framework, including dependencies and configurations.

System Tools

System Tools Module

The system tools module provides a collection of useful utilities for system administration and maintenance. It includes tools for disk management, file operations, security, and more.

Key Commands:
lebit tools disk
Provides disk management tools, including partitioning, formatting, and mounting operations.
lebit tools security
Security tools for system hardening, firewall configuration, and security auditing.
lebit tools backup
Tools for creating, managing, and restoring backups of your system and data.
lebit tools network
Network management tools, including network troubleshooting, configuration, and monitoring.

Mining Tools

Mining Tools Module

The mining tools module provides tools for setting up and managing cryptocurrency mining operations. It supports various cryptocurrencies and mining algorithms, with optimized configurations for different hardware.

Key Commands:
lebit mining setup [crypto]
Sets up mining software for a specific cryptocurrency, with optimized configurations for your hardware.
lebit mining start
Starts the mining process with the configured settings.
lebit mining stop
Stops the mining process while preserving your configurations.
lebit mining stats
Displays mining statistics, including hashrate, power consumption, and earnings.
