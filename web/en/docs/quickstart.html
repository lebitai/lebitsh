Quick Start Guide

Installation

Installing Lebit.sh is simple and straightforward. Just run the following command in your terminal:

curl --proto '=https' --tlsv1.2 -sSf https://lebit.sh | sh

This command downloads and executes the installation script, which will guide you through the setup process.

Basic Usage

After installation, you can use Lebit.sh by running the lebit command in your terminal. Here are some basic commands to get you started:

View Help

lebit --help

Displays the help information with all available commands and options.

Update Lebit.sh

lebit update

Updates Lebit.sh to the latest version.

List All Modules

lebit list

Lists all available modules and their current installation status.

Installing Modules

Lebit.sh is designed with a modular architecture. You can install only the modules you need. To install a module, use the following command:

lebit install [module_name]

For example, to install the Docker module:

lebit install docker

Running Module Commands

Each module has its own set of commands. To run a module-specific command, use the following format:

lebit [module_name] [command]

For example, to get system information using the system module:

lebit system info

Configuration

Lebit.sh can be configured to suit your needs. The configuration file is located at ~/.config/lebit/config.yaml. You can edit this file directly or use the configuration commands:

lebit config set [key] [value]

For example, to set the default installation path:

lebit config set install.path /opt/lebit
