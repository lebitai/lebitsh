What is Lebit.sh?

Lebit.sh is a Linux system initialization toolkit that provides a suite of tools to help you quickly and efficiently set up and configure your Linux environment. Whether you're a system administrator, developer, or power user, Lebit.sh can save you time and effort in configuring your Linux systems.

Getting Started

After installing Lebit.sh, you can run it directly from your terminal:

Terminal
lebit

This will open the interactive menu where you can select the category of tools you want to use.

Configuration

Lebit.sh comes with a powerful configuration system. You can customize its behavior through configuration files:

  • User configuration: ~/.config/lebitsh/config.conf
  • System configuration: /etc/lebitsh/config.conf
  • Default configuration: /etc/lebitsh/defaults.conf

You can manage configuration through the "Configuration" menu option in Lebit.sh or by directly editing these files.

Logging

Lebit.sh provides detailed logging to help with troubleshooting. Logs are stored in /tmp/lebitsh.log by default, but this can be configured in the configuration files. You can view logs through the "View Logs" menu option in Lebit.sh.
