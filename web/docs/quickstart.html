快速入门指南

快速上手Lebit.sh的基本功能和命令

简介

Lebit.sh是一个Linux系统初始化工具包，提供一系列模块化工具，帮助您快速设置和配置Linux环境。本指南将引导您完成安装和基本使用流程。

安装

Lebit.sh的安装非常简单，只需一行命令即可开始使用：

curl --proto '=https' --tlsv1.2 -sSf https://lebit.sh | sh

这个命令将下载并运行交互式安装程序，您可以选择需要的模块进行安装。

注意

安装前请确保您有sudo权限，因为某些模块可能需要安装系统软件包。

基本用法

安装完成后，您可以通过以下方式使用Lebit.sh：

启动主菜单

运行 lebit 命令启动交互式主菜单：

lebit

从主菜单中，您可以选择所需的模块。

直接调用特定模块

您可以直接调用特定模块，例如Docker模块：

lebit docker

使用模块特定命令

每个模块都有自己的子命令，例如：

lebit docker install

运行特定模块的安装子命令。

可用模块

Lebit.sh包含以下主要模块：

模块名称 命令 描述
系统管理 lebit system 系统管理和信息收集工具
Docker管理 lebit docker Docker和Docker Compose的安装和管理
开发环境 lebit dev 开发环境设置工具
系统工具 lebit tools 实用系统工具集合
挖矿工具 lebit mining 加密货币挖矿工具

配置

Lebit.sh的配置文件位于 ~/.config/lebit/config。您可以编辑此文件来调整Lebit.sh的行为。

有关更多配置选项，请参阅配置指南。

获取帮助

如果您需要帮助，可以使用以下命令：

lebit help

或者使用特定模块的帮助命令：

lebit docker help
