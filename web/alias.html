Quick Alias Setup

Enhance your terminal productivity with our collection of useful aliases. Run the following command to automatically set up these aliases in your shell configuration:

curl -s https://lebit.sh/alias | bash

How it works:

  1. The command downloads and executes a script from our server
  2. The script adds useful aliases to your shell configuration file (~/.bashrc or ~/.zshrc)
  3. Aliases are immediately available in new terminal sessions
  4. For current session, you might need to run: source ~/.bashrc (or ~/.zshrc)
