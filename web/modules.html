功能模块

深入了解Lebit.sh提供的各种功能模块及其详细特性

系统管理

系统管理模块提供一系列工具，帮助您管理和优化Linux系统，收集系统信息，并执行常规维护任务。

curl -sSf https://lebit.sh/install | sh -s -- system

硬件信息收集

快速获取系统的CPU、内存、磁盘和网络信息，生成格式化报告用于系统评估。

系统清理与深度清理

删除临时文件、缓存和日志，释放磁盘空间，并清理不再需要的包和旧内核。

时间同步工具

配置NTP服务，确保系统时钟与标准时间服务器保持同步，对于日志记录和时间敏感的应用至关重要。

Docker管理

Docker管理模块简化了Docker和Docker Compose的安装、配置和管理过程，帮助您快速开始使用容器化技术。

curl -sSf https://lebit.sh/install | sh -s -- docker

Docker安装

自动安装最新版本的Docker和Docker Compose，包括所有必要的依赖项，适用于各种Linux发行版。

Docker升级

轻松将现有Docker安装升级到最新版本，确保获得最新的功能和安全修复。

Docker配置

根据最佳实践配置Docker守护程序，包括存储驱动、日志轮转和网络设置。

资源清理

清理未使用的容器、镜像、网络和卷，释放磁盘空间，保持Docker环境整洁。

容器监控

提供工具监控Docker容器的资源使用情况，包括CPU、内存和网络使用率。

开发环境

开发环境模块帮助您快速设置完整的开发环境，安装流行的编程语言和工具，节省时间专注于编码。

Golang安装

安装最新版本的Go编程语言，配置GOPATH和环境变量，准备好开始Go开发。

Node.js (通过NVM)

Rust安装

SQLite3安装

快速别名
